Postman Collection: Lektoři

1. Lektori all

        GET https://czechitas-api.vercel.app/api/czechitas/lektori/all
        Description: Retrieve all lecturers.

2. Lektori random

        GET https://czechitas-api.vercel.app/api/czechitas/lektori/random
        Description: Retrieve a random lecturer.

3. Lektori id

        GET https://czechitas-api.vercel.app/api/czechitas/lektori/3
        Description: Retrieve lecturer with a specific ID.

4. Lektori name

        GET https://czechitas-api.vercel.app/api/czechitas/lektori/:name
        Description: Retrieve lecturer by name.

Additional API Documentation

Postman Collection: Ukol - Veronika Obrtelová

1. Add - Záporná čísla

        GET https://czechitas-api.vercel.app/api/math/add?a=-3&b=-4
        Description: Adds two negative numbers. Expects correct sum.

2. Add - Mix čísel

        GET https://czechitas-api.vercel.app/api/math/add?a=3&b=-4
        Description: Adds one positive and one negative number. Expects correct sum.

3. Add - Neplatné vstupní hodnoty

        GET https://czechitas-api.vercel.app/api/math/add?a=-3&b=msdksfgmkdf
        Description: Tests handling of invalid input values. Expects bad request.

4. Add - Sčítání výsledků několikrát po sobě

        GET https://czechitas-api.vercel.app/api/math/add?a=3&b=3
        Description: Tests addition with repeated results. Expects correct sum after multiple calls.

5. Add - Nulové hodnoty

        GET https://czechitas-api.vercel.app/api/math/add?a=0&b=0
        Description: Adds two zeros. Expects zero.

6. Add - Kladná čísla

        GET https://czechitas-api.vercel.app/api/math/add?a=3&b=4
        Description: Adds two positive numbers. Expects correct sum.

7. Add - Desetinné místo

        GET https://czechitas-api.vercel.app/api/math/add?a=abc&b=3
        Description: Tests decimal numbers and non-numeric input. Expects correct result or error.

8. Add - Iracionální čísla

        GET https://czechitas-api.vercel.app/api/math/add?a=√2&b=π
        Description: Tests irrational numbers. Expects bad request.

9. Add - Chybějící parametr

        GET https://czechitas-api.vercel.app/api/math/add?a=3&b
        Description: Tests missing parameter. Expects error.

10. Add - další

        GET https://czechitas-api.vercel.app/api/math/add?a=\"abc\"&b=3
        Description: Tests with incorrect parameter values. Expects error.

11. Add - Plus sign

        GET https://czechitas-api.vercel.app/api/math/add?a=+&b=4
        Description: Tests addition with plus sign. Expects correct result.

12. Add - Minus sign

        GET https://czechitas-api.vercel.app/api/math/add?a=-&b=4
        Description: Tests addition with minus sign. Expects bad request.

13. New Request

        GET https://czechitas-api.vercel.app/api/math/add?a=\"abc\"&b=3
        Description: Tests escaped characters. Expects error.
